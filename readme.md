## Laravel School Management System

This school managemen system is built on top of Laravel Framework. With the help of some awesome packages. Some of the packages used are-
1. Entrust
2. Confide
3. Ardent
4. Former
5. Bootstrap
6. Basset
7. Carbon
8. Datatables
9. vespakoen/menu
10. And tons of more
